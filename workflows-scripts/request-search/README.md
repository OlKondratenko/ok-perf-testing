## Workflow steps:

##### Workflow for requests_searchByOpenStatus.jmx

Search request by open status

1. Open Requests App
2. Select Request status drop down and select all open statuses:
- Open - Awaiting delivery
- Open - Awaiting pickup
- Open - In transit
- Open - Not yet filled
